10.0 - Creating a Module

run the cli command:

 ng generate module pokemon-base

it will create a folder <pokemon-base> with an file <pokemon-base.module.ts>

10.1 - Creating a component

run this cli command:

ng generate component pokemon-base/pokemon-list --module=pokemon-base/pokemon-base.module.ts

now export the component in pokemon-base.module decorator:

  @NgModule{{'({'}}
    declarations: [PokemonListComponent],
    imports: [CommonModule],
    exports: [PokemonListComponent],
  '}}

10.2 - Import the new module into app.module.ts

  @NgModule{{'({'}}
    declarations: [AppComponent],
    imports: [BrowserModule, AppRoutingModule, FormsModule, PokemonBaseModule],
    providers: [],
    bootstrap: [AppComponent],
  '}}
