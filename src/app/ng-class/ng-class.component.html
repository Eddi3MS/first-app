8 - ngClass and [class]

works like css modules. Accessing classes inside the css file with dot notation, but it is only applied if the condition that follows evaluate to true. 

  [class.isCool]="pokemon.isCool"

With ngClass, the class is the 'key', and the value is what will be evaluated to apply or not the class, like: 

  [ngClass]="{{ "{" }}isCool: pokemon.isCool{{ "}" }}",
note: you can apply different classes based in different conditions with comma (,) like an normal object with keys and values, only the keys are classes that will be applied if the values evaluate to true, like: 
  [ngClass]="{{"{"}}isntCool: !pokemon.isCool, cool: pokemon.isCool {{ "}" }}"
