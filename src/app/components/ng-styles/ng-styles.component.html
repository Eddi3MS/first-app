9 - ngStyle and [style]

accessing the propertie style, like [style] and passing css like normal inline css: 

  [style]=" !pokemon.isCool ? 'background: blue; color: white' : ''"

or accessing a property in style, like:

  [style.background]=" !pokemon.isCool ? 'blue' : 'red'"

And you can do it with ternary, like:

  [style]=" !pokemon.isCool ? 'background: blue; color: white' : 'background: red'"

using ngStyle is more like the react.js style inline syntax:

  [ngStyle]="{{'{'}}'backgroundColor': pokemon.isCool ? 'red' : 'blue'{{'}'}}"
